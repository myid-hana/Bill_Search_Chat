# 🖥️ Bill Search Chat
keyword 를 입력하여 keyword 와 관련된 법안을 검색할 수 있는 채팅입니다. 
주로 flutter 를 이용하여 만든 웹 서비스입니다.  


라이브러리를 쓰지 않고 직접 구현하는 애니메이션을 더 연습하고 싶어서 라이브러리 사용을 최소화하며 다양한 애니메이션을 넣으려 노력하고 있습니다.
클린 코드를 공부하며 적용하고 연습하고 있습니다.
